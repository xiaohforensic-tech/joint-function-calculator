# 关节功能丧失辅助计算

面向法医临床司法鉴定工作的离线辅助计算 PWA，支持肩、肘、腕、髋、膝、踝六个四肢大关节、颈椎、腰椎，以及手功能评分。

## 启动

在本目录执行：

```powershell
python -m http.server 8000
```

然后在浏览器打开 `http://127.0.0.1:8000/`。首次打开后可通过浏览器“安装应用”作为桌面小程序使用。

也可直接打开根目录中的 `关节功能丧失辅助计算-离线单文件.html`，无需服务器或网络。

## 计算范围

- 人体损伤程度鉴定：查表法（活动度 × 肌力），伤侧与健侧同法计算后相减。
- 人体损伤致残程度鉴定：查表法或方向均分法；界面提示查表法的限定适用范围。
- 膝关节查表法：屈曲、伸展查表值相加并封顶 100%。
- 肘关节查表法：自动换算为屈曲 90° 中立位。
- 肩关节旋转默认采用贴壁位，参考值为外旋 45°～60°、内旋 45°～70°。
- 颈椎、腰椎：按 SF/T 0111—2021 附录 A.6 的六方向正常参考值采用方向均分法，不自动对应四肢鉴定阈值。
- 同轴另一方位按 100%：由鉴定人依据实际活动弧人工勾选并进入记录。
- 中立位 0° 法：不能达到中立位录入负值，过伸录入正值；肘、膝按有符号屈伸活动弧计算。
- 非功能位强直：记录关节状态，并对致残程度鉴定中的相应直接条款作提示。
- 手功能活动度：录入拇指 MCP/IP 及其余手指 MCP/PIP/DIP 实测活动弧，自动显示与参考活动弧的比例及 C.8 分档。
- 损伤程度手功能：按附录 C.7 固定权重、部分实体缺失长度公式及 4%/16%/36% 阈值辅助计算；普通活动度下降不直接线性折算为 C.7 权重。
- 致残程度手功能：按附录 C.8、图 C-1 及表 C-10 计分，双手按 `A+B×(200-A)/200` 加权；标准正文直接条款优先。

## 隐私与记录

无服务器、无上传。历史记录仅保存在浏览器 localStorage；可导出 JSON、复制计算意见、打印或另存为 PDF。

## 重要提示

本工具只提供辅助量化与记录，不替代原始检验记录、伤病关系分析、标准适用判断或正式司法鉴定意见。
